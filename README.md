# ai_bee_helper-v1
Этот репозиторий отражает мой прогресс в изучении ai engeneering. 
Здесь я использую OpenAI Complitions API через Openrouter.

Здесь есть асинхронные функции, но по факту в этом проекте они не раскрываются.
Я использую pydantic для проверки формата ответов и try/except, чтобы всё не ломалось.
Есть работа с тулами. Пока один тул - заглушка (get_hive_status(hive_id)), но он правильно отрабатывает.
Есть лёгкая защита на уровне систем промта - не общаться на темы не связанные с пчеловодством.
